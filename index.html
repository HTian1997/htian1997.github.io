Research Interest

My research mainly lies in machine learning and molecular dynamics simulations.
Currently, I focus on the following topics:

  • Decipher protein allostery through machine learning based methods;
  • Machine learning and database based allostery development.

Projects

getarticle, an open source Github repository   

  • A package based on SciHub and Google Scholar that can download articles given DOI, website address or keywords;
  • Imported in Python or used as command line.


PASSer: Protein Allosteric Sites Server    passer.smu.edu

  • A web server to predict allosteric sites given protein PDB ID or PDB file;
  • Implemented using Django framework and JSmol;
  • Learn both physical properties through XGBoost model and pocket topology through graph convolutional network.
